User Manual

Getting the most out of CamKeeper.

CamKeeper saves your iRacing driving view — FOV, driver height, shift horizon, and virtual mirror — separately for every car, and puts the right one back so you never re-tune when you switch series.

1Installing

CamKeeper is a single portable program — there is no installer. Download CamKeeper.exe and put it anywhere you like (a folder in your Documents or on your desktop is fine). Double-click it to run.

Windows SmartScreen: because CamKeeper is a new, unsigned app, Windows may show a "Windows protected your PC" prompt the first time. Click More info → Run anyway. This is expected for small independent tools and gets less frequent as the app becomes known.

2First launch

The first time you run CamKeeper it shows the terms of use and asks you to agree before continuing. This is a one-time step; you will only see it again if the terms change in a future version.

3The main window & tray

CamKeeper runs quietly in your system tray (near the clock). Double-click the tray icon, or right-click it for the menu, to open the window.

The window shows the car iRacing currently reports, the live view values it reads from your app.ini, and whether the applied values match the saved profile for that car. The tray menu also lets you apply a saved profile, open the editor, view release notes, check for updates, and read the "How it works" guide.

4Saving a profile

Tune a car's view in iRacing the way you like it. Then, in CamKeeper, use Save current as profile to store those values under that car. Repeat for each car you drive. Each car keeps its own profile.

5Applying a profile

Pick the car you're about to drive and choose Apply. CamKeeper writes that car's saved values into app.ini, ready for your next session.

Important — how iRacing reads settings: iRacing only loads app.ini when a session starts, and ignores changes made while you're already on track. So apply your profile before you join the session (while you're in the iRacing UI). It then loads with the car. Applying mid-session has no effect until the next session.

6Editing offline

The Manage Profiles editor works even when iRacing is closed. Open it to view and hand-edit any car's saved values, copy a setup from one car to a similar one as a starting point, or fix a value. Changes are written to iRacing the next time that car is applied — the editor never touches app.ini while you're idle.

7Backups & restore

Every time a profile is saved, CamKeeper keeps a timestamped backup (the most recent ten per car). If a value gets changed by accident, open a car's backups and restore an earlier snapshot in one click. Backups are plain, human-readable files.

8Import & export

You can export a single car's profile to a file to share it, or import one someone sends you. You can also export your entire library to one archive and import it back — handy when moving to a new PC or keeping a safe copy.

9Auto-save & auto-apply

Two optional toggles, both off by default:

  • Auto-apply on car change — CamKeeper applies the saved profile automatically when it detects a new car, instead of you clicking Apply. (Remember the session-start rule above.)
  • Auto-save — CamKeeper saves the current live values to that car's profile automatically. Leave this off if you don't want an in-session tweak to overwrite a saved profile by accident.

10Start with Windows

Turn on Start with Windows in settings and CamKeeper will launch to the tray when you log in, ready in the background whenever you race.

11Updating

CamKeeper checks for new versions and, when one is available, tells you and asks before updating — it never replaces itself silently. You can also check manually from the tray menu. Release notes for each version are viewable in the app.

12Uninstalling

There's no uninstaller because there's nothing installed. To remove CamKeeper completely:

1. Deletethe CamKeeper.exe file
2. Deletethe folder %appdata%\CamKeeper (paste that into the address bar of File Explorer to find it) — this holds your profiles, backups, and settings

Your iRacing files are left exactly as they are.

13Troubleshooting

My saved view didn't load. Make sure you applied the profile before joining the session — iRacing only reads the file at session start.

CamKeeper shows the wrong car. Car detection can briefly lag a real car change by about a second; give it a moment.

The values in CamKeeper look stale. Use the refresh in the app; CamKeeper reads app.ini from disk, and iRacing only writes its own changes back when it exits a session.

Windows blocked the app. See the SmartScreen note under Installing — choose More info → Run anyway.
